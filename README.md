Программа представляет собой датчик, собирающий показания температуры. Датчик считывает N измерений в диапазоне от -50 до 50°C.

Для хранения данных используются динамические массивы. Среди данных содержатся максимум, минимум и среднее из полученных температур. Один из массивов хранит в себе показания, выше среднего.

Для работы с массивами используются указатели и арифметика указателей.

В конце программы производится освобождение памяти и обнуление указателей.

### Пример вывода:
```
=== Анализ сенсора ===
Введите количество измерений: 6
Введите 6 значений (от -50 до 50): 10 15 8 -2 18 20

Среднее значение: 11.5
Минимум: -2
Максимум: 20

Значения выше среднего:
15 18 20
```
